# Lost & Found ID System

A web-based application for managing lost and found university ID cards.  
Currently, the project includes the **user interface for reporting lost/found IDs** and a **home page for search/retrieval**.

## Project Status
This project is **work in progress**.  
Only the front-end web interface has been implemented so far with no styling.  
Additional features (image recognition, database integration, admin verification, etc.) will be added gradually.

## Features (so far)
- Lost ID reporting form
- Found ID reporting form
- Home page displaying list of lost and found IDs

## Tech Stack
- HTML, CSS, JavaScript (frontend)
- Flask (backend, planned)
- SQLite (database, planned)

## Roadmap
- [ ] Connect forms to backend (Flask)
- [ ] Integrate SQLite database
- [ ] Add image recognition for ID matching
- [ ] Implement admin verification and notifications
